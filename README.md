# fe_no2_assignment
부산대학교 프론트엔드 심영찬입니다!

vercel배포 상의 문제로 main 브랜치가 아닌 context 브랜치가 default가 되어 있으며,
바로 prop-drilling브랜치로 개발을 시작했기 때문에 main 브랜치에는 init말고는 아무 내용도 없습니다!
모두 같은 내용으로 작성했습니다!
